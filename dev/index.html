ClimFlowsTestCases.TestCase — Type
abstract type TestCase end

Parent type for test case types. Derived type must implement a constructor accepting a named tuple of parameters and the function default_params. For example:

TC = Williams91{6}
params = default_params(TC)
case = TC((params..., R0=3e-3))

ClimFlowsTestCases.testcase — Method
case = testcase(Case::Type, F=Float64 ; params...)

Returns a test case with default parameters for case Case with floating-point precision F. Optional params override the default parameters for the test case. For example:

w91_6 = testcase(Williamson91{6}) # Float64, default parameters
w91_6 = testcase(Williamson91{6}, Float32 ; R0=1e-3) # Float32, overrides R0 (planetary radius)
